# Student Performance Prediction using Artificial Neural Networks

## Overview

This project applies **Artificial Neural Networks (ANN)** to analyze and predict student academic performance based on behavioral and academic factors.

The objective is to understand how variables such as **study habits, attendance, and prior academic performance** influence final exam outcomes.

Two predictive models were developed:

- **Regression Model** ‚Äî predicts the student's exam score
- **Classification Model** ‚Äî predicts the student's performance category (grade)

---

## Dataset

The dataset contains **6607 student records** describing various aspects of academic behavior and environment.

### Features Used

- Hours Studied
- Attendance
- Sleep Hours
- Previous Scores
- Tutoring Sessions
- Physical Activity
- Extracurricular Activities

### Target Variables

- **Exam Score** _(for regression model)_
- **Performance Grade** _(for classification model)_

Grades are generated by dividing exam scores into **four categories using quartile-based grouping**.
